\doxysubsection{Detailed Description}
Provides functionality for evaluating mathematical expressions in Reverse Polish Notation (\doxylink{classRPN}{RPN}). 

This class includes methods to print supported operations, check if a character is a valid operation, and evaluate \doxylink{classRPN}{RPN} expressions. It is designed for use in an \doxylink{classRPN}{RPN} calculator.
